# ReqTracker MVP

Первый шаг web-приложения для отслеживания требований ТЗ.

## Как открыть

Откройте `index.html` в браузере. Приложение работает как статическая страница и хранит загруженные требования в `localStorage` браузера.

## Что уже есть

- загрузка требований из `.xlsx`, `.xls` и `.csv`;
- генерация демонстрационного набора требований;
- таблица реестра требований;
- поиск по коду, тексту, владельцу и источнику;
- фильтр по статусу;
- сводные счетчики по требованиям.

## Поддерживаемые колонки импорта

Приложение автоматически сопоставляет русские и английские заголовки:

- код: `code`, `id`, `код`, `номер`, `идентификатор`;
- текст: `text`, `description`, `requirement`, `текст`, `описание`, `требование`;
- статус: `status`, `state`, `статус`, `состояние`;
- приоритет: `priority`, `prio`, `важность`, `приоритет`;
- владелец: `owner`, `assignee`, `author`, `владелец`, `ответственный`, `автор`;
- источник: `source`, `document`, `origin`, `источник`, `документ`.

Для чтения Excel в браузере используется SheetJS CDN.
